Variables in javascript

We can create variables in 3 ways in java srcipt

  1. using let keyword
  2. using const keyword
  3. using var keyword

Identifier rules

  1. Identifier cannot be start with or use special characters(only '$', '_' are allowed)
  2. Idenitifer cannot be start with numbers (but can use number)
  3. Identifier cannot be same as keyword
  4. Identifiers can any character between 'A' to 'Z' and 'a' to 'z'
