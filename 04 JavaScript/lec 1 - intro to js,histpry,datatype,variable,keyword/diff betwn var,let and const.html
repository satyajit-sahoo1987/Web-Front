Diff betwn var,let and const

  1. let & const are block scoped where as var is function or global scope
  2. let & var can be reassigned,where as const can't be reassigned
  3. Redeclaration is posssible in case of var keyword where as redeclaration is not possible incase of let &const keyword
  4. if we try to access the variable ddeclared with var keyword before their declaration then these variable values will be undefined ,where as if we try to access variable declared with let & const before there declaration then it will generate reference Error(because values are present inside TMP-Temporal Dead Zone)
